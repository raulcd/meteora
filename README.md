Meteora
-------
A performance testing web application made in Python

Installation
------------

Note: Meteora requires Python 3.4

```
mkvirtualenv meteora
pip install -e .
```

Running tests
-------------

First, install tox:

```
sudo pip install tox
```

Then run:

```
tox
```
